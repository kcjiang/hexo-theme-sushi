# hexo-theme-sushi

A highly functional theme that adapts to the reader's preferences. Let them read, search, navigate, and more with ease. Completely free and fully responsive, released under the MIT license.

**Demo: https://jkc.icu**



# Instructions

1. [Download this theme](https://github.com/kcjiang/hexo-theme-sushi.git)
2. Unzip the file to Hexo's `themes` folder.
